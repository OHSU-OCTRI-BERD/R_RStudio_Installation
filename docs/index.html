Pre-workshop installation

  • Please install the software before the workshop.
    • The workshop will begin assuming you already have R & RStudio set up.
  • There will be an installation help session starting 30 minutes before the first day of the workshop in case you are having difficulty installing R & RStudio and want to ask for help.

Install R & RStudio on your computer

  • R & RStudio are two separate programs. You must first install R, and then install RStudio desktop.
  • If you prefer tutorials or videos with installation instructions, please see the next section below.

Step 1: Install R

Note: as of 4/28/2026, the latest version of R is 4.6.0. If you are reading these instructions later, references for version 4.6.0 below will be a later version.

  • Windows
    • Go to https://cran.rstudio.com/bin/windows/base/ and download R using the link “Download R-4.6.0. for Windows”.
  • Mac OS
    • Go to https://cran.rstudio.com/bin/macosx/ and download the latest .pkg file (currently R-4.6.0.pkg) listed on the left column based on your computer’s specifications.
    • For some packages and uses, Xcode and XQuartz may be required, but there is no need to install these until you need them (Xcode takes a very long time to install).
      • They will not be needed for the workshop.

Step 2: Install RStudio Desktop

  • Go to https://posit.co/download/rstudio-desktop/#download
  • Go to step 2: Install RStudio at the bottom right of the page, and select your computer’s operating system.
  • Click the blue box to download the installation file corresponding to your operating system.
    • The blue box should say something like “DOWNLOAD RSTUDIO DESKTOP FOR MACOS 14+” (or Windows).
  • Follow the download instructions on your computer.

Step 3: Open RStudio

  • After installing both R & RStudio, open RStudio by clicking on the RStudio icon, a round blue ball with R in it, that should look something like this:

  • Do not open the regular R software, it runs in the background while you use RStudio.

FAQ

  1. What if I already have R & RStudio installed? Do I need to install them again to the latest version?.
    • R: If you already have R installed, make sure it is at least version 4.1.0 in order to use the packages that we will be using during the workshop. Just follow the R installation instructions above to update everything.
      • Note that you will need to reinstall your R packages when you update R. We recommend making a list of your R packages before updating R.
      • An example of how to this is described here.
      • If you are a beginner, I recommend just installing R packages as you need them.
    • RStudio: RStudio updates are released regularly, and we recommend updating to the latest version.
      • Updates are quick to install and do not require reinstalling R packages.
      • Follow the RStudio installation instructions above to update RStudio.
  2. What if I can’t install R & RStudio on my computer before the workshop?
    • If you are using an older computer or computer for which you do not have administrative rights, then you might run into problems installing R & RStudio.
    • If you cannot install R & RStudio before the workshop, we highly recommend creating a free Posit Cloud account to use RStudio online for the workshop.
